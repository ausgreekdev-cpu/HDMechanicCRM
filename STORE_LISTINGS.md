## Deployment Checklist

### Microsoft Store
- [ ] Create Microsoft Partner Center account ($19 one-time)
- [ ] Build PWA with manifest.json and service worker
- [ ] Package as MSIX using PWABuilder or manual
- [ ] Submit with screenshots and descriptions
- [ ] Wait for certification (typically 24-48 hours)

### Google Play Store
- [ ] Create Google Play Developer account ($25 one-time)
- [ ] Wrap PWA as TWA using Bubblewrap
- [ ] Generate signing key
- [ ] Build AAB file
- [ ] Submit with screenshots and descriptions
- [ ] Wait for review (typically 3-7 days)
